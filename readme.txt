#NOTICE
AI Makers Kit - audio driver add-on package
This package is tested on kernel of "RASPBIAN Stretch 2017-09-07 Version"
Please check RASPBIAN version before installation.

Follow below step
1. copy packages to any directory under home directory of raspbian (ex, /home/pi/install)
2. Add executing permission to shell files, if they don't have permissions to execute
   (aimk-install.sh after-install.sh)
3. execute install script file 
   -->  "> ./aimk-install.sh"
4. Wait complete installation and reboot with user's prompt
5. After reboot execute audio setup script
   -->  "> ./after-install.sh"
6. Wait end of set-up and it will reboot with user's prompt 
7. Enjoy AI Makers Kit
